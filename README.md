# Experiência Digital de Relacionamento Financeiro com IA Generativa
Este projeto é a entrega final para o Desafio de Projeto da DIO (Digital Innovation One). Consiste em um assistente virtual inteligente focado em relacionamento financeiro, integrando a potência do modelo Google Gemini 2.5 Flash com lógica de programação em Python para oferecer uma experiência de usuário (UX) clara, segura e contextualizada.

## Tecnologias Utilizadas

Linguagem: Python 3.13
IA Engine: Google Gemini 2.5 Flash (via SDK google-genai)
Bibliotecas: google-genai, python-dotenv (recomendado)

## Objetivos do Desafio

IA Generativa: Implementar compreensão de linguagem natural para dúvidas financeiras.
Relacionamento: Criar uma persona de assistente financeiro prestativo.
Funcionalidades: Incluir FAQs inteligentes e simulações de cálculos financeiros.
UX: Oferecer interações claras e tratamento de erros amigável.
