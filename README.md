# Meu Assistente Financeiro - Projeto Final Bradesco/DIO

Este é o meu projeto de conclusão do curso da DIO. A ideia aqui foi sair do básico e criar um assistente que realmente entenda de finanças e ajude o usuário de um jeito simples e direto. Usei o Google Gemini 2.5 Flash integrado com Python para fazer a "mágica" acontecer.

## O que eu usei
Python 3.13: A base de toda a lógica.

Google Gemini 2.5 Flash: IA por trás das respostas.

Biblioteca google-genai: Para conectar o código com o modelo da Google.

## O que o projeto entrega
A ideia central foi focar na experiência de quem está usando. O assistente não só responde perguntas, mas:

Entende de verdade: Consegue bater um papo sobre investimentos e dúvidas financeiras de forma natural.

Tem "personalidade": Configurei o bot para ser um consultor prestativo, evitando respostas genéricas demais.

Faz simulações: Além de conversar, ele ajuda com cálculos e explicações de produtos financeiros.

É resiliente: O código foi pensado para não travar se a internet falhar ou se o limite de mensagens for atingido.

## Como eu resolvi os problemas
Durante o desenvolvimento, precisei ajustar o código para lidar com as versões mais novas do Python e as limitações de cota da API. Isso garantiu que o assistente fosse mais "robusto" e não apenas um chat simples que para de funcionar do nada.
